# 3OTOL WebSite | NTI Project

A responsive website built by converting a **Figma design** into HTML & CSS.

## 🌐 Live Demo
🔗 [Live Preview](https://ahmedtika74.github.io/3OTOL-WebSite/)

## 📌 Project Overview
This project demonstrates the ability to transform a Figma design into a fully responsive website using **HTML5 and CSS3**. The layout adapts to mobile, tablet, and desktop screens, focusing on clean UI and accurate implementation of the design.

## 🛠 Technologies Used
- HTML5 – Semantic structure
- CSS3 – Styling, layout, and responsiveness
- Figma – Original design reference

## 🧠 Features
✔ Accurate implementation of the Figma design  
✔ Fully responsive and mobile-first  
✔ Organized folder structure for CSS and images
